# KK's Koala Tycoon: Developer Commands

Use these commands in the Roblox chat to test game mechanics quickly.

| Command | Usage | Description |
| :--- | :--- | :--- |
| **`/reset`** | `/reset` | Performs a "Hard Reset": clears all progress, resets exhibits, gives the basic hammer, and brings back the Head Vet. |
| **`/skip`** | `/skip` | Skips to late-game: grants $10,000 and marks tutorial as complete. |
| **`/spawnkoala`** | `/spawnkoala [stage] [rarity]` | Spawns a koala at your feet.<br>• **Stage:** 1 (Newborn) to 5 (Adult).<br>• **Rarity:** `cute`, `extra`, or `ultra`. |
| **`/growkoala`** | `/growkoala` | Instantly advances all koalas in your current exhibit to the next stage. |
| **`/cash`** | `/cash [amount]` | Grants the specified amount of cash. |

---

### Growth Stages Reference
1. **Newborn:** 0.2x size, stationary.
2. **6 Month:** 0.4x size, stationary.
3. **1 Year:** 0.7x size, waddles.
4. **2 Year:** 0.9x size, climbs.
5. **Adult:** 1.0x size, full AI, generates revenue.

### Rarity Reference
- **Cute:** Default look.
- **Extra:** Gold Glow.
- **Ultra:** Purple Glow + Sparkles.
